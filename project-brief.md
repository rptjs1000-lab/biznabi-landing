# 비즈나비 (BIZNABI) — 에이전시 홈페이지 기획서

## 프로젝트 개요
- **프로젝트명:** 비즈나비 공식 홈페이지
- **목적:** 영업/마케팅용 랜딩 페이지 — 잠재 고객(매장 사업주)이 서비스를 이해하고 상담을 신청하도록 유도
- **타입:** 싱글 페이지 (원페이지 랜딩)

## 대상 사용자
- 동물병원, 미용실, 필라테스, 음식점/카페, 뷰티, 병원 등 오프라인 매장 사업주
- IT에 익숙하지 않은 소상공인 → 쉽고 명확한 메시지 필요

## 브랜드 아이덴티티
- **회사명:** 비즈나비 (BIZNABI)
- **로고:** 나비 모양 심볼 + "비즈나비" 텍스트 (가로 배열)
- **브랜드 컬러:** 네이비(#0B2545) ~ 블루(#1B6AAA) ~ 라이트블루(#4AADCF) 그라디언트
- **슬로건:** "매장 운영, 나비처럼 가볍게" 또는 "당신의 매장에 날개를 달아드립니다"
- **톤앤매너:** 신뢰감 + 현대적 + 친근함 (B2B이지만 소상공인 대상이라 딱딱하지 않게)

## 페이지 구성 (섹션)

### 1. Hero 섹션
- 비즈나비 로고 + 핵심 슬로건
- "무료 상담 신청" CTA 버튼
- 매장 운영 관련 배경 이미지 (서비스업 분위기)

### 2. 문제 제기 (Pain Point)
- 매장 사업주들이 겪는 문제 3가지 제시
  - "예약 전화 놓치고 계세요?"
  - "노쇼 때문에 매출 손실?"
  - "홈페이지가 없어서 검색에 안 나오나요?"

### 3. 솔루션 소개
- 비즈나비가 제공하는 해결책 개요
- 홈페이지 + 예약 자동화 + 알림톡 자동 발송을 한 번에

### 4. 서비스 패키지
- **예약 자동화 패키지** — 홈페이지 불필요, 예약 위젯 + 대시보드 + 알림톡
- **홈페이지 + 자동화 패키지** — 위 패키지 + 맞춤 홈페이지
- 각 패키지별 포함 기능 리스트

### 5. 주요 기능 상세
- 온라인 예약 위젯 (링크만으로 예약 접수)
- 매장 관리 대시보드 (예약/고객/방문 관리)
- 알림톡 자동 발송 (예약 확인, 리마인드, 후속 관리)
- 맞춤 홈페이지 (업종별 최적화)

### 6. 대상 업종
- 동물병원, 미용실, 필라테스/요가, 음식점/카페, 뷰티/네일, 병원/의원
- 업종 아이콘 + 간단한 설명

### 7. 도입 프로세스
- 상담 신청 → 요구사항 분석 → 제작/설정 → 교육 → 운영 시작
- 4~5단계 스텝

### 8. 숫자로 보는 비즈나비 (신뢰도)
- 제작 소요 기간, 예약 자동화율, 노쇼 감소율 등 수치 (마케팅용)

### 9. FAQ
- 자주 묻는 질문 3~5개

### 10. CTA + 상담 신청 폼
- 업종, 매장명, 연락처, 관심 패키지 선택
- "무료 상담 신청" 버튼

### 11. Footer
- 회사 정보, 연락처, 소셜 링크

## 기술 스택
- Next.js 14 + React 18 + TypeScript + Tailwind CSS 3.4
- 정적 랜딩 페이지 (API Route: 상담 신청 폼 전송용 1개)
- Vercel 배포

## MVP 범위
- 싱글 페이지 랜딩
- 상담 신청 폼 (이메일 전송 또는 Supabase 저장)
- SEO 기본 설정
- 반응형 (모바일 퍼스트)
