# 비즈나비 (BIZNABI) 비즈니스 모델

*문서 버전: 2.0 / 최종 수정: 2026-04-21*
*이전 버전(v1.2)은 3라인 구조 / v2.0에서 2라인으로 재편 + 기업 자동화는 보류 라인으로 이동*

---

## 1. 회사 개요

- **회사명:** 비즈나비 (BIZNABI)
- **슬로건:** "당신의 비즈니스에 날개를 달아드립니다"
- **미션:** 소상공인이 디지털 도구를 쉽고 저렴하게 도입할 수 있도록 맞춤형 자동화 솔루션을 제공한다.
- **핵심 가치:** 신뢰, 맞춤, 자동화, 접근성

### 운영 방식
- **타깃 시장**: 소상공인 (1차) — 홈페이지 + 예약·고객·알림 자동화 필요한 업종
- **영업 채널**: 비즈나비 랜딩, 크몽, 당근, 소상공인 커뮤니티, 지인 소개
- **운영 인력**: 대표(본인) + 남편분(공동 개발) + Claude (AI 어시스턴트)

---

## 2. 서비스 라인업 (2라인)

### 2-1. 홈페이지 제작
**일회성 단발 결제**. 업종 맞춤 반응형 홈페이지.

**기본 구성 (5p 이내):**
- 업종 맞춤 디자인
- 반응형 (모바일·태블릿·PC)
- 상담 폼 + 카카오톡 상담 버튼
- 기본 SEO (메타/사이트맵/OG)
- 수정 2회 보장

**옵션 (필요 시 추가):**
- 페이지 확장, 예약 위젯, 블로그/공지 CMS, 갤러리, 후기 시스템, 심화 SEO, 로고 디자인, 다국어, 결제 연동, 쇼핑몰 전환 등
- 상세 옵션 가격표: [pricing-plan-v1.md](../docs/pricing-plan-v1.md) 참조

**납품 기간:** 7~10 영업일

### 2-2. 비즈나비 자동화 패키지
**월 구독형**. 예약·고객·알림 자동화 + 모듈별 확장.

**기초 구성 (월 9.9만원~):**
- 온라인 예약 위젯
- 예약 달력/대시보드
- 알림톡 기본 발송 (예약 확정, 리마인드)
- 고객 DB 기초
- 월 알림톡 500건 포함

**모듈 (선택 추가):**
- 고객 CRM 고도화, 포인트·적립금, 쿠폰/기프티콘, 리뷰 수집, 재방문 자동 알림, 매출 리포트, 상품·메뉴 관리, 카카오 챗봇 등
- 상세 모듈별 월 구독료: [pricing-plan-v1.md](../docs/pricing-plan-v1.md) 참조

**초기 세팅비:** 무료 (AI 자동화로 원가 거의 없음)
**제작 기간:** 3~5 영업일
**최소 약정:** 없음, 월 단위 해지 가능

### 2-3. (보류) 기업 맞춤 자동화
**현 라인업 제외.** 인바운드 문의 발생 시 별도 상담으로 처리.

사유: 영업·맞춤 개발 사이클이 소상공인 라인과 완전히 다름. 낭만클럽 집중 기간 동안 리소스 배분 어려움. 낭만클럽 마무리 후 **별도 라인(비즈나비 엔터프라이즈 등)** 신설 검토.

## 3. 가격 체계

### 3-1. 공개 가격 (랜딩페이지 노출)

| 라인업 | 시작가 | 구조 |
|--------|-------|------|
| 홈페이지 제작 | **80만원~** | 일회성 |
| 비즈나비 자동화 패키지 | **월 9.9만원~** | 월 구독 |

### 3-2. 번들 할인

**홈페이지 제작 + 자동화 패키지 동시 계약 시**
- 홈페이지 초기비 **20% 할인**
- 예: 80만원 → **64만원** + 월 9.9만원~
- 3개월 이내 자동화 해지 시 할인 차액 반환

### 3-3. 상세 견적 기준

**상세 가격 테이블은 홈페이지에 노출하지 않음.** 운영용 문서 `[pricing-plan-v1.md](../docs/pricing-plan-v1.md)` 참조.

상담 시 업종·페이지 수·필요 기능·모듈 조합 확인 후 맞춤 견적.

### 3-4. 공통 옵션

| 항목 | 가격 |
|------|------|
| 호스팅·도메인 관리 | 월 2만원 |
| 도메인 구매 대행 | 1회 +2만원 + 도메인 비용 |
| 납품 후 수정 | 건당 10~30만원 |
| 연간 유지보수 (월 2회 수정·백업) | 월 3만원 |

## 4. 수익 모델

### 4-1. 수익 구조

```
[초기 수익]
- 홈페이지 제작비 (1회): 80~300만원 (옵션 조합 따라)

[반복 수익 (MRR)]
- 자동화 월 구독: 9.9~25만원 / 고객사
- 알림톡 초과 발송 과금
- 추가 모듈 활성화 시 구독료 상향
- 호스팅·유지보수 월비
```

| 30개 | 15만원 | 450만원 | 5,400만원 |

| 100개 | 15만원 | 1,500만원 | 1억 8,000만원 |

### 4-3. 홈페이지 제작 수익 보조

홈페이지 단발 매출은 현금흐름 주축. 평균 150만원·월 2~3건 목표 → 월 300~450만원 추가 수익.

## 5. 타겟 고객

### 5-1. 1차 타깃: 예약·고객관리 니즈가 큰 소상공인
- 미용실, 필라테스/요가, 뷰티/네일, 동물병원, 카페, 음식점, 학원, 청소·출장 서비스 등
- 특징: IT 비숙련, 예약·고객 관리에 시간 소모, 홈페이지 부재 or 업데이트 어려움
- 니즈: 저렴하고 쉬운 디지털 전환

### 5-2. 2차 타깃: 홈페이지 단발 제작 고객 (자동화 불필요)
- 인테리어·건설 업체, 사무·전문 서비스업, 개인 브랜드 등
- 특징: 포트폴리오·회사 소개 홈페이지 필요, 자동화 니즈 낮음
- 니즈: 신뢰도 있는 전문 홈페이지 1회 제작

### 5-3. 보류: 중소기업 업무 자동화 고객
- 현재는 영업 X. 문의 시 별도 상담.
- 낭만클럽 마무리 후 별도 라인(엔터프라이즈)으로 재개

| 일반 웹에이전시 | 200~500만+ | 10~30만 | X | 고단가, 자동화 별도 |
| **비즈나비** | **80만~** | 월 9.9만~ (선택) | **O (선택 연계)** | **단발 + 구독 유연 조합** |

### 6-2. 예약·고객관리 SaaS 시장

| 서비스 | 초기비 | 월비용 | 맞춤 세팅 | 알림톡 | 홈페이지 |

| 네이버 예약 | 무료 | 수수료 3.19% | X | X | X |

| 캐치테이블 | 무료 | 5.5~11만 | X | 제한 | X |

| **비즈나비** | **무료** | **9.9만~** | **O (업종별)** | **O (풀커스텀)** | **O (연계 가능)** |

### 6-3. 비즈나비 차별점

1. **홈페이지 + 자동화 연계** — 한 파트너에서 해결, 업셀 번들 할인
2. **모듈형 구독** — 필요한 기능만 골라서 월 구독, 사용 안 하는 기능 비용 X
3. **맞춤 세팅** — 범용 SaaS와 달리 업종/매장 운영 방식에 맞춰 개별 설정
4. **알림톡 풀커스텀** — 템플릿·발송 시점·후속 메시지 자유 설정
5. **수수료 없음** — 거래 수수료 기반이 아닌 정액 월 구독 (거래 많은 매장에 유리)

## 7. 손익분기 분석

### 7-1. 자동화 패키지 (월 구독)

**고객사 1개당 원가 (기초 패키지 기준):**
- 초기 세팅: AI 자동화로 인건비 거의 없음 (~0원)
- 월 서버비 (Supabase·Vercel 분담): ~1만원
- 알림톡 발송 원가 (500건): ~1만원
- CS 인건비: ~1만원

**월 수익:** 9.9만원
**월 원가:** ~3만원
**월 이익:** ~6.9만원
**손익분기:** 1개월차부터 흑자

### 7-2. 홈페이지 제작 (일회성)

**고객사 1개당 원가 (기본 패키지 80만원 기준):**
- AI 활용 제작: ~20~30만원 (툴 사용료·디자인 리소스·검토 시간)
- 초기 수익: 80만원 - 30만원 = **50만원 이익**

**번들 할인 적용 시 (64만원):**
- 수익: 64만원 - 30만원 = **34만원 이익** + 자동화 월 6.9만원 이익 (장기 누적)

## 8. 계약 및 운영 정책

### 8-1. 계약 조건
- **최소 약정 기간:** 없음 — 월 단위 이용, 언제든 해지 가능
- **자동화 패키지:** 초기비 무료, 월 구독만 결제
- **홈페이지 제작:** 계약 시 50%, 납품 시 50%
- **번들 계약:** 홈페이지 20% 할인 + 자동화 최소 3개월 구독 약속

### 8-2. 고객 교육 방식
- **사용 가이드 영상** (기본 제공, 재사용)
- **화상 미팅** (추가 요청 시 30분~1시간)
- **CS 지원** (카카오톡·전화 상시)

### 8-3. 고객사 관리 구조
- **비즈나비 운영자** → `admin.biznabi.com` 에서 전체 고객사 관리 (예정)
- **고객사(사장님)** → 자기 홈페이지 `/admin` 에서 본인 매장 관리
- 홈페이지 없는 자동화만 고객 → 관리자 플랫폼에서 직접 관리

### 8-4. 제작 기간
| 라인 | 제작 기간 |
|------|----------|
| 홈페이지 제작 | 7~10 영업일 |
| 자동화 패키지 | 3~5 영업일 |
| 번들 동시 계약 | 7~10 영업일 (병행 작업) |

## 9. 영업 전략

### 9-1. 고객 확보 채널
1. 랜딩페이지 SEO + 구글·네이버 검색 유입 (biznabi.com)
2. **크몽 상품 등록** — 단발 홈페이지 80만원 포지션 (평균가 일치)
3. 인스타그램·블로그 콘텐츠 마케팅
4. 소상공인 커뮤니티 (카페, 단톡방, 스레드)
5. 기존 고객 레퍼럴 (추천 할인 정책)
6. 낭만클럽 공급사·파트너 네트워크 (식품 B2B 교차 판매)

### 9-2. 상담 접수 → 응대 플로우

```
[유입]
랜딩페이지 상담 폼 / 카카오톡 채널 / 크몽 메시지

        ↓

[자동 처리]
1. Supabase contact_inquiries 테이블 저장
2. 카카오톡 챗봇이 초기 응대 (예정)
3. 고객에게 자동 안내: "접수 완료, 1영업일 내 연락드리겠습니다"
4. 운영자(본인)에게 카카오톡 알림 전송

        ↓

[수동 처리]
5. 운영자가 admin 대시보드에서 상담 목록 확인
6. 전화·카톡으로 1:1 상담 (업종·필요 기능·예산 파악)
7. 맞춤 견적서 발송 (pricing-plan-v1.md 기반)

        ↓

[계약 → 온보딩]
8. 계약 체결 + 결제 (홈페이지 50% 선결제)
9. 제작/세팅 (3~10 영업일)
10. 사용 가이드 영상 안내 → 운영 시작
```

### 9-3. 리텐션 전략
- 월 1회 운영 리포트 발송 (예약 건수, 노쇼율, 고객 증감)
- 분기 1회 모듈 추가 제안 (업셀)
- 알림톡 템플릿 최적화 컨설팅
