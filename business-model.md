# 비즈나비 (BIZNABI) 비즈니스 모델

---

## 1. 회사 개요

- **회사명:** 비즈나비 (BIZNABI)
- **슬로건:** "당신의 비즈니스에 날개를 달아드립니다"
- **미션:** 소상공인과 중소기업이 디지털 도구를 쉽고 저렴하게 도입할 수 있도록 맞춤형 자동화 솔루션을 제공한다.
- **핵심 가치:** 신뢰, 맞춤, 자동화, 접근성

---

## 2. 서비스 라인업

### 2-1. 예약 자동화 패키지
홈페이지 없이도 바로 사용 가능한 예약 자동화 솔루션

**포함 기능:**
- 온라인 예약 위젯 (링크 공유 방식, 카카오톡/인스타 등에 삽입)
- 매장 관리 대시보드 (예약/고객/방문 현황)
- 알림톡 자동 발송 (예약 확인, 전일 리마인드, 방문 후 후속 메시지)
- 고객 데이터 관리 (연락처, 방문 이력, 메모)

**초기 세팅 작업 (AI 자동화, 인건비 거의 없음):**
- Supabase 고객사 계정/DB 생성
- 서비스 메뉴, 영업시간, 휴무일 등록
- 알림톡 템플릿 설정 및 발송 규칙 세팅
- 사용 가이드 영상 제공

**제작 기간:** 3~5 영업일

### 2-2. 홈페이지 + 자동화 패키지
업종별 맞춤 홈페이지와 예약 자동화를 한 번에 제공

**포함 기능:**
- 예약 자동화 패키지의 모든 기능
- 업종별 맞춤 홈페이지 (반응형, 모바일 최적화)
- SEO 기본 최적화 (메타데이터, 사이트맵, 구조화 데이터)
- 도메인 연결 및 SSL 설정

**초기 제작 작업:**
- 요구사항 분석 및 콘텐츠 기획
- 업종별 디자인 시안 제작 → 확인 → 개발
- 예약 자동화 전체 세팅
- QA 및 배포
- 사용 가이드 영상 제공

**제작 기간:** 7~10 영업일

### 2-3. 업무 자동화 패키지
중소기업의 반복 업무를 맞춤 프로그램으로 자동화

**가능 범위:**
- 견적서/발주서 자동 생성
- 매출/업무 리포트 자동화
- 재고/발주 관리 시스템
- 업무 프로세스 맞춤 개발
- 관리자 대시보드

## 3. 가격 체계

### 3-1. 랜딩페이지 표시 가격 (시작가)

| 패키지 | 초기비 | 월 유지비 |
|--------|--------|----------|
| 예약 자동화 | **무료** | **월 9.9만원~** |
| 홈페이지 + 자동화 | **200만원~** | **월 15만원~** |
| 업무 자동화 | 맞춤 견적 | 맞춤 견적 |

> * 부가세 별도
> * "~"는 기본 사양 기준이며, 추가 기능에 따라 변동

### 3-2. 실제 상담 시 세부 견적 기준

| 항목 | 가격 | 비고 |
|------|------|------|
| 홈페이지만 제작 (기본 랜딩페이지) | 150만원~ | 원페이지, 반응형, SEO 포함 |
| 홈페이지 + 예약 자동화 | 200만원~ | 홈페이지 + 예약위젯 + 알림톡 + 대시보드 |
| 추가 기능 (게시판, 갤러리 등) | 별도 견적 | 기능 복잡도에 따라 산정 |
| 추가 자동화 (리뷰 관리, 재방문 유도 등) | 별도 견적 | 모듈 단위 추가 |

### 3-3. 월 유지비에 포함되는 항목

| 항목 | 예약 자동화 (9.9만원~) | 홈페이지+자동화 (15만원~) |
|------|----------------------|-------------------------|
| 서버 호스팅 (Vercel/Supabase) | O | O |
| 알림톡 기본 발송량 | O (월 500건) | O (월 500건) |
| 시스템 유지/업데이트 | O | O |
| CS 지원 (카카오톡/전화) | O | O |
| 홈페이지 콘텐츠 수정 | - | O (월 2회) |
| SSL/도메인 관리 | - | O |

> * 알림톡 기본 발송량 초과 시 건당 10~15원 별도 과금

## 5. 타겟 고객

### 5-1. 1차 타겟: 예약 기반 매장 (소상공인)
- 동물병원, 미용실, 필라테스/요가, 음식점/카페, 뷰티/네일, 병원/의원
- 특징: IT 비숙련, 예약 관리에 시간 소모, 노쇼 문제, 홈페이지 부재
- 니즈: 저렴하고 쉬운 디지털 전환

### 5-2. 2차 타겟: 중소기업 (업무 자동화)
- 제조업, 무역/수출입, 유통/도매, 건설/인테리어
- 특징: 반복 업무 과다, 엑셀 의존, 시스템 부재
- 니즈: 맞춤 업무 자동화, 리포트 자동화

## 8. 계약 및 운영 정책

### 8-1. 계약 조건
- **최소 약정 기간: 없음** — 월 단위 이용, 언제든 해지 가능
- 예약 자동화: 초기비 무료, 월 구독만 결제
- 홈페이지+자동화: 초기 제작비 1회 + 월 유지비
- 부가세 별도

### 8-2. 고객 교육 방식
- **사용 가이드 영상** (기본 제공): 한 번 제작 → 모든 고객사 재사용
- **화상 미팅** (추가 요청 시): 줌/구글밋으로 30분~1시간
- **CS 지원** (상시): 카카오톡/전화로 운영 중 지속 지원

### 8-3. 고객사 관리 구조
- **비즈나비 운영자** → admin.biznabi.co.kr에서 전체 고객사 관리
- **고객사(매장 사장님)** → 자기 홈페이지 /admin에서 본인 매장만 관리
- 홈페이지 없는 고객(예약 자동화만)은 관리자 플랫폼에서 직접 관리

### 8-4. 제작 기간
| 패키지 | 제작 기간 |
|--------|----------|
| 예약 자동화 | 3~5 영업일 |
| 홈페이지 + 자동화 | 7~10 영업일 |
| 업무 자동화 | 상담 후 결정 |

> 기간은 여유 있게 안내. 조기 완료 시 신뢰도 UP, 지연 시에도 약속 범위 안.

## 9. 영업 전략

### 9-1. 고객 확보 채널
1. 랜딩페이지 SEO + 구글/네이버 검색 유입
2. 인스타그램/블로그 콘텐츠 마케팅
3. 소상공인 커뮤니티 (카페, 단톡방)
4. 기존 고객 레퍼럴 (추천 할인)

### 9-2. 상담 프로세스
1. 랜딩페이지에서 상담 신청 폼 접수
2. 전화/카카오톡 1차 상담 (니즈 파악)
3. 맞춤 견적서 발송
4. 계약 → 제작/세팅 → 교육 → 운영 시작

### 9-3. 리텐션 전략
- 월 1회 운영 리포트 발송 (예약 건수, 노쇼율, 고객 증감)
- 분기 1회 기능 업데이트/개선 제안
- 알림톡 템플릿 최적화 컨설팅
